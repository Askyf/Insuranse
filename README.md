## 📋 Σύνοψη

Μια **μοντέρνα, πλήρως responsive ιστοσελίδα** για την παρουσίαση και προώθηση ασφαλιστικών προϊόντων. Η ιστοσελίδα είναι σχεδιασμένη με βάση τις σύγχρονες αρχές UI/UX και προσφέρει εξαιρετική εμπειρία χρήστη σε όλες τις συσκευές.

## ✨ Χαρακτηριστικά

### ✅ Βασικά Χαρακτηριστικά
- **Fully Responsive Design** - Βελτιστοποιημένη για mobile, tablet και desktop
- **Modern UI/UX** - Σύγχρονο design με animations και transitions
- **Δυναμική Φόρτωση Προϊόντων** - Από JSON αρχείο
- **Φίλτρο Προϊόντων** - Ανά κατηγορία με live filtering
- **FAQ Accordion** - Expand/collapse συχνών ερωτήσεων
- **Φόρμα Επικοινωνίας** - Με validation και form submission
- **Smooth Scrolling** - Για καλύτερη πλοήγηση
- **Mobile Navigation** - Burger menu για κινητές συσκευές

### 🚀 Προχωρημένα Χαρακτηριστικά
- **Lazy Loading** - Βελτιστοποίηση απόδοσης
- **Debounced Events** - Για καλύτερη απόδοση
- **Intersection Observer** - Για animations on scroll
- **Form Validation** - Πλήρης client-side validation
- **Notifications System** - User-friendly ειδοποιήσεις
- **Performance Optimized** - Βέλτιστη ταχύτητα φόρτωσης
